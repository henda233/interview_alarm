# 复试通告自动抓取程序：复试警报

定时爬取目标院校通告网页，判断是否发布复试通告，并即时以电子邮件的方式告知用户。

## 技术栈

python作为基础编程语言。

请求库采用requests，解析库采用BeautifulSoup4.

邮件发送库采用标准库smtplib

## 项目功能与实现细节

### 功能1：定时爬取目标院校通告网页并判断是否有复试通告

通过网页的最新通告发布日期是“2026-03-13”，而本项目认为，一旦出现发布日期不是“2026-03-13”的通告（即一定是新发布的通告），则判定复试通告发布。

#### 网页HTML布局

通告网页（目标网页）URL：https://cs.gzu.edu.cn/16270/list.htm

网页是静态HTML网页，没有采用前端技术。

通告存储在标签`<ul class="wp_article_list"> `下，每个通告为`<li class="list_item i1">`，其中`i1`表示第一个通告，显然第二个通告为`<li class="list_item i2">`，以此类推。

在每个`<li class="list_item ix">`标签下，有如下标签：

```html
     <li class="list_item i1"> 
         <div class="fields pr_fields"> 
             <span class="Article_Index">1</span> 
             <span class="Article_Title"><a href="/2026/0313/c16270a265715/page.htm" target="_blank" title="贵州大学计算机科学与技术学院2026年电子信息专业博士研究生招生“申请-考核”制实施办法">贵州大学计算机科学与技术学院2026年电子信息专业博士研究生招生...</a></span> 
         </div> 
         <div class="fields ex_fields"> 
              
             <span class="Article_PublishDate">2026-03-13</span> 
              
         </div> 
     </li> 
```

可知，`<span class="Article_Title"></span>`存储通告标题，`<span class="Article_PublishDate">2026-03-13</span> `存储通告发布时间。

#### 定时设定

每过`check_time`时间爬取一次网页判断是否有复试通告，时间单位分钟。`check_time`作为全局变量，可在项目落地后自由修改，默认为15分钟。

### 功能2: 电子邮件推送

如果功能1判断有复试通告，立刻通过预设的电子邮箱向用户邮箱发送通知，其标题设置为`[IA]疑似复试通告发布！`，内容设置为通告标题。

#### 心跳推送

为了防止程序异常关闭，保证程序持续运行，每隔`live_time`时间向用户邮箱发送信息，其标题设置为`[IA]心跳反馈`，内容设置为`这是心跳包。`。`live_time`单位是分钟，默认为3个小时。

通过上面的心跳推送，可以确保程序异常关闭后，即时通知用户。

#### 运行时间

程序只在早上8点到晚上9点运行，其余时间暂停运行，不发送任何邮件包括心跳推送。

### 功能3: json配置文件

功能1和功能2中的`check_time`、`live_time`、SMTP地址、SMTP端口、邮箱账号、授权码、用户邮箱地址和基准日期（即判断是否发布复试通告的比对日期）存储在json配置文件中。

除此之外，目标URL硬编码在代码中。

### 其他功能

需要伪造请求头，预防反爬虫。

## 项目开发流程

一切功能封装为函数。不需要实现日志功能。

### 配置文件函数

读取配置文件，如果不存在自动创建并采用默认值。

返回配置字典。

### 爬取解析函数

输入配置字典。

伪造请求头，请求并获取目标网页HTML，解析获取第一条通告的发布日期和标题。

根据配置字典的基准日期判断第一条是否为最新通告，并将其视为复试通告。

如果是复试通告，将配置字典中的基准日期更新为最新日期，则新的配置字典，标题和发布日期，否则返回None。

### 邮件发送函数

输入配置字典，邮件标题和邮件正文。

登录邮箱，向目标用户邮箱地址发送相应的标题和正文。

### 判断当前是否为工作时间函数

判断当前时间是否在早上8点到晚上9点，如果是则返回真，否则为假。

### 主循环函数

接收配置字典。

这是主程序，主循环。

实现定时爬取判断、发送心跳包和非工作时间暂停运行。

### 主函数

首先运行配置文件函数获取配置字典，然后进入主循环函数。
